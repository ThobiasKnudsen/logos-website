Easy and powerful

The tools you need without the complexity you don't.

f

Just type math

Write x² + 3*x - 7 and see results instantly. No imports, no boilerplate, no compilation step. Unicode symbols like π, ∂, ∫ work natively.

∫

Symbolic algebra built in

Differentiate, integrate, and simplify expressions symbolically — built in, not bolted on. Results appear as you type.

▶

GPU-accelerated

Code compiles to WGSL shaders and runs on your GPU automatically. Write a parallel for loop and it dispatches to compute shaders.

■

Real-time visualization

Every expression renders live in the output pane. Fractals, surfaces, Monte Carlo paths — all animated and interactive with zoom and pan.

Py

Python SDK

Use Logos from Python with @logos and @logos_parallel decorators. CPU JIT, GPU dispatch, zero-copy NumPy, and data ingress (CSV, SQL, HTTP, and more).

λ

Full programming language

Functions, loops, conditionals, arrays, and block expressions. Enough to write Monte Carlo simulations, solvers, and custom shaders from scratch.
